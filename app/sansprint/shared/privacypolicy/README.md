### Privacy Policy for Number Sprint

Last updated: July 4, 2026

Number Sprint does not collect, store, use, or share any personal or sensitive user data.

The application works entirely offline and does not require an internet connection. It does not use user accounts, analytics services, advertising services, tracking technologies, or third-party services that collect user data.

Any game progress, scores, records, or settings are stored locally on the user’s device and are not transmitted to the developer or any third party.

Number Sprint does not request access to location, contacts, photos, microphone, camera, or other sensitive device information.

When the application is deleted, all locally stored application data is removed according to the operating system’s standard behavior.
